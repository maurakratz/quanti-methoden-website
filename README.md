# Quantitative Methoden mit R & R-Studio

Dieses Repository enthält die Materialien für das Seminar 
**Quantitative Methoden mit R & R-Studio** am Otto-Suhr-Institut 
der Freien Universität Berlin (SoSe 2026).

## Inhalt

- `folien/` – Seminarfolien (Quarto/revealjs)
- `uebungen/` – Übungsaufgaben und Lösungen
- `scripts/` – R-Skripte
- `data/` – Datensätze (nicht versioniert)
- `docs/` – Gerenderte Website (GitHub Pages)
